This guide is for clinicians using the **ET TransformX** system, designed for those familiar with EdgeTaper, EdgeX7, Race Evo, and other .04/.06 taper systems.

## Prerequisites

- Established access cavity
- Working length determined
- Glide path confirmed to working length

## Recommended Settings

Use your standard ET-style motor settings. No changes required from your existing EdgeTaper or similar workflows.

## Sequence

Follow the sequence outlined in the technique workflow PDF for your specific case anatomy.

### Key Points

1. **Always irrigate** between files
2. **Recapitulate** with a small hand file after each rotary
3. **Re-irrigate** before proceeding to the next file
4. **Confirm patency** throughout the procedure

## Avatar™ Tip Advantage

The Avatar tip design allows the file to:
- Navigate canal curvatures with less transportation
- Maintain centred preparation
- Reduce the risk of ledging or perforation
