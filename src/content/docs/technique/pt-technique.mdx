This guide is for clinicians using the **PT TransformX** system, designed for those experienced with ProTaper Gold or ProTaper Next systems.

## Prerequisites

- Established access cavity
- Working length determined
- Glide path confirmed to working length

## Recommended Settings

Use your standard ProTaper-style motor settings. No changes required from your existing ProTaper Gold or ProTaper Next workflows.

## Sequence

Follow the progressive shaping sequence outlined in the technique workflow PDF.

### Key Points

1. **Always irrigate** between files
2. **Recapitulate** with a small hand file after each rotary
3. **Re-irrigate** before proceeding to the next file
4. **Confirm patency** throughout the procedure

## Avatar™ Tip Advantage

The Avatar tip design allows the file to:
- Follow the progressive shaping philosophy with enhanced control
- Maintain centred preparation through curvatures
- Reduce the risk of transportation
