This guide is for clinicians using the **PT TransformX** system, designed for those experienced with ProTaper Gold or ProTaper Next systems.

## Prerequisites

- Established access cavity
- Working length determined
- Glide path confirmed to working length

## Motor Settings

| Step | File | Speed | Torque | Purpose |
|------|------|-------|--------|---------|
| 1 | SX | 300 RPM | 3.0 Ncm | Coronal Shaping |
| 2 | S1, S2 | 300 RPM | 3.0 Ncm | Middle Third Shaping |
| 3 | F1, F2, F3 | 300 RPM | 3.0 Ncm | Apical Finishing |

## Step-by-Step Sequence

### Step 1: Coronal Shaping
- Use **SX** at 300 RPM, 3.0 Ncm
- Create initial coronal flare
- Establish straight-line access to the mid-root
- Irrigate thoroughly

### Step 2: Middle Third Shaping
- Use **S1** at 300 RPM, 3.0 Ncm — shape to resistance
- Irrigate → Recapitulate → Re-irrigate
- Use **S2** at 300 RPM, 3.0 Ncm — shape to working length
- Irrigate → Recapitulate → Re-irrigate

### Step 3: Apical Finishing
- Use **F1** at 300 RPM, 3.0 Ncm — to working length
- Select **F2** or **F3** based on apical anatomy
- Complete apical sizing

## Key Points

1. **Always irrigate** between files
2. **Recapitulate** with a small hand file after each rotary
3. **Re-irrigate** before proceeding to the next file
4. **Confirm patency** throughout the procedure
5. **Follow the progressive philosophy** — each file prepares for the next

## Avatar™ Tip Advantage

The Avatar tip design allows the file to:
- Follow the progressive shaping philosophy with enhanced control
- Maintain centred preparation through curvatures
- Reduce the risk of transportation
- Support controlled termination in the optimal apical zone

## File Selection Guide

| File | Tip Size | Taper | Use Case |
|------|----------|-------|----------|
| SX | 19 | Progressive | Coronal flaring |
| S1 | 17 | Progressive | Initial shaping |
| S2 | 20 | Progressive | Full-length shaping |
| F1 | 20 | .07 | Small canals |
| F2 | 25 | .08 | Medium canals |
| F3 | 30 | .09 | Larger canals |
